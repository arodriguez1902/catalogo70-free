Lightweight plugin to create SEO-optimized product catalogs with customizable fields and smart CTAs.

== Description ==

Catalogo70 is a lightweight and powerful plugin for creating product catalogs in WordPress. Designed to be fast, intuitive, and SEO-optimized.

= Features =

* **Customizable fields** - Create custom fields for your products (text, number, select, date, file, URL, email)
* **Unlimited categories** - Organize your products in categories and subcategories
* **SEO optimized** - Schema.org markup, meta tags, Open Graph, and microdata to improve your ranking
* **Smart CTAs** - Customizable call-to-action buttons per product
* **Image gallery** - Multiple images per product with drag & drop support
* **Import/Export** - Import and export products in CSV format
* **Search and filters** - Search and filter system by custom fields
* **Responsive** - Mobile and tablet-friendly design
* **Shortcodes** - Easy integration in any page or post

= Free version limits =

* Products: up to 10
* Categories: up to 10
* Custom fields: up to 5
* Images per product: up to 3

= Premium =

* Unlimited products, categories, and fields
* Advanced Import/Export
* Unlimited CTAs
* Advanced SEO
* Priority support

== Installation ==

1. Sube la carpeta `catalogo70` al directorio `/wp-content/plugins/`
2. Activa el plugin desde el menú 'Plugins' en WordPress
3. Ve a `Catálogo Pro > Campos` para configurar tus campos personalizados
4. Ve a `Productos > Añadir nuevo` para crear tu primer producto
5. Usa el shortcode `[catalogo70_productos]` para mostrar el catálogo en cualquier página

== Frequently Asked Questions ==

= ¿Cómo muestro el catálogo en una página? =

Usa el shortcode `[catalogo70_productos]` en cualquier página o entrada.

= ¿Puedo personalizar los campos de los productos? =

Sí, desde `Catálogo Pro > Campos` puedes crear campos personalizados de diferentes tipos.

= ¿Cómo activo la licencia Premium? =

Ve a `Catálogo Pro > Licencia` e ingresa tu license key.
